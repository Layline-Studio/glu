# Glu

Monorepo for the Glu app.

| Folder | What | Stack |
|---|---|---|
| [`web/`](web/) | Landing page / website (deployed on Cloudflare Pages) | Astro + React + Tailwind |
| [`mobile/`](mobile/) | Mobile app | Flutter |

## Web

```sh
cd web
npm install
npm run dev
```

## Mobile

```sh
cd mobile
flutter pub get
flutter run
```
